# Punto de Partida

Sistema integral de ingeniería pedagógica diseñado para transformar información académica en experiencias de aprendizaje altamente efectivas.

## ¿Qué es Punto de Partida?

Punto de Partida no es una plataforma, un curso ni una metodología aislada. Es un sistema completo de transformación educativa cuyo propósito es convertir conocimiento complejo en comprensión profunda, accesible y duradera.

## Misión

Diseñar experiencias educativas capaces de convertir conocimiento complejo en comprensión profunda, accesible y duradera para cualquier estudiante.

## Principios fundamentales

1. **Centralidad del estudiante** — toda decisión favorece el aprendizaje del estudiante.
2. **Comprensión antes que memorización** — comprender tiene prioridad sobre recordar.
3. **Actividad antes que pasividad** — el aprendizaje requiere participación activa.
4. **Claridad antes que complejidad** — la complejidad innecesaria es una falla de diseño.
5. **Progresión antes que saturación** — el conocimiento se construye gradualmente.
6. **Evidencia antes que intuición** — las decisiones pedagógicas se fundamentan en evidencia.
7. **Calidad antes que velocidad** — la rapidez nunca compromete la efectividad educativa.
8. **Mejora continua** — todo componente del sistema puede ser perfeccionado.
